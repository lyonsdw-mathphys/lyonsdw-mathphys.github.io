Instructor's solution for Exercise 1.
  1. This sequence fits a linear pattern with \(a=2\) and \(d=4\text{.}\) We can see this by checking that the pattern fits.
    \begin{equation*} 2,\; 6=2+4, \; 10=6+4,\; \ldots \end{equation*}
  2. This sequence fits an exponential pattern with \(a=2\) and \(r=3\text{.}\) We can see this by checking that the pattern fits.
    \begin{equation*} 2,\; 6=2\cdot 3, \; 18=6\cdot 3,\; \ldots \end{equation*}
  3. This sequence cannot be linear or exponential. If the sequence were linear, then \(d\) would have to be \(6-2=4\text{,}\) but then the third term would have to be \(6+4=10\text{.}\) If the sequence were exponential, then \(r\) would have to be \(6/2=3\text{,}\) but then the fourth term would have to be \(18\cdot 3=54\text{.}\)
